# Images

Put your static assets here (recommended formats: **WebP**, **PNG**, or **JPG**).

Examples:

- `public/images/mitul-portrait.jpg` -> use `src="/images/mitul-portrait.jpg"` with `next/image`
- `public/images/project-acme-dashboard.png` -> use `src="/images/project-acme-dashboard.png"`

Tip: if you rename files, update the matching `src` in `app/page.tsx` (and `components/*` if you add more).
